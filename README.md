# Kudipeer Wallet Landing Page

## How to run
- On dev
```sh
npm run start
```

## Classes
- class name should be descriptive
- Do not overide the styling of `.container`
- container defaults to
```css
<!-- mobile -->
.container {
	padding-left: 5%;
	padding-right: 5%;
}

<!-- Larger screens -->
.page-wrapper {
	padding-left: 2.5%;
	padding-right: 2.5%;
}
```
